# CANiTEXT911 Front-End

Main Vue.js-powered landing page for [CANiTEXT911.us](https://canitext911.us).

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
